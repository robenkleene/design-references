# Photoshop Layers

- `⌘J`: Duplicate layer
- `⇧⌘N`: New layer
- `⇧⌘⌥N`: New layer with default name
- `⌘[` / `⌘]`: Send layer backward / forward
- `⌘⇧[` / `⌘⇧]`: Send layer to back / front
- `⌥[` / `⌥]`: Select next / previous layer
- `⌫`: Delete a layer
- `⌥`-click eye icon: Toggle on this layer and off all other layers
