# Pixelmator Layers

- `⌥⌘,`: Toggle hidden
- `⌥⌘A`: Select all layers
- `⌥⌘E`: Merge selected layers
- `⌥⇧⌘E`: Merge all layers
- `⌥click` a layer: Hide other layers

## Select

- `⌥[` / `⌥]`: Select next / previous layer
- `⌥.` / `⌥,`: Select top / bottom layer

## Move

- `⌘[` / `⌘]`: Move selected layer backward / forward
- `⌥⌘[` / `⌥⌘]`: Move selected layer to the back / front
